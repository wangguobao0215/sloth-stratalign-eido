## What is StratAlign?

StratAlign is a **strategic alignment accelerator for management consultants**. Through industry playbook pre-fills, a structured questionnaire engine, and AI-Native Potential Scoring, it boosts consultant efficiency 3-5x in data collection, analytical modeling, and report writing for digital transformation strategy engagements.

## Key Features

### Six Modules
1. **Industry Detection & Setup** (Module 0) — Auto-load industry playbook, three consultant modes (Full / Accelerated / Expert)
2. **Strategic Decoding** — BSC four-perspective objectives + contradiction detection + strategy hypothesis register (If-Then-Because)
3. **Bottleneck Scan** — Pain point identification + L1-L2 capability decomposition + root cause vs symptom classification
4. **AI-Native Potential Triage** — Four-dimension weighted scoring + per-scenario data coefficient + sensitivity analysis
5. **Coverage Analysis** — IT mapping + application portfolio health quadrant + capability heat map
6. **Investment Portfolio** — Full ROI calculation (NPV + payback) + three-scenario analysis + risk register

### What's New in v1.0.0
- **Consultant-First Positioning**: Optimized as a management consultant efficiency tool
- **AI Scoring Model**: Pure multiplication → 4D weighted model (Rule 25% + Data 30% + Frequency 20% + Implementation Complexity 25%), 0-100 scale
- **Industry Playbook Library**: Manufacturing / Retail / Financial Services / Healthcare / Professional Services
- **Report Template**: Fill-in-blank → narrative structure + Executive Summary + risk scenario analysis
- **Methodology Enhancements**: Strategy hypothesis register + L1-L2 capability decomposition + app portfolio health quadrant + ROI calculation template
- **Questionnaire Engine**: Module 0 industry detection + three consultant modes + pre-fill defaults + completeness validation

## AI Scoring Formula (v1.0)

```
Weighted Raw = 0.25×Rule + 0.30×Data + 0.20×Frequency + 0.25×Complexity
Normalized = (Weighted Raw - 1) / 4 × 100    (0-100 scale)
Final Score = Normalized × Scenario Data Coefficient (0.4 / 0.6 / 0.8 / 1.0)

Tiers: ≥75 Immediate | 50-74 Short-term | 25-49 Foundation First | <25 Defer
```
